Verwendung von Trigger

create or modify trigger before_insert_person

when event

on t trigger_type

execute stored_procedure;

Erstelle oder modifiziere ein Trigger

When

  • before- löst aus bevor Event stattfindet
  • after- löst aus nachdem Event stattfindet

event

  • insert- löst beim einfügen aus
  • update- löst daten verändern aus
  • delete- löst beim löschen aus

trigger_type

  • for each row
  • for each statement

create trigger before_insert_person

before insert

on person before each row

execute stored_procedure;
